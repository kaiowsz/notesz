## :vulcan_salute: Getting started

#### Cloning the repository

```bash
$ git clone https://github.com/kaiowsz/notesz.git

$ cd notesz
```
#

#### Installing dependencies

```bash
$ yarn
```
or
```bash
$ npm install
```

#

#### Running the application

```bash
$ npm dev
```
or
```bash
$ yarn dev
```
